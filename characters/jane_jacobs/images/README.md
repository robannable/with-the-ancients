# Jane Jacobs Portrait

Please add a portrait image named `jane_jacobs.jpg` to this directory.

Recommended image: A photograph of Jane Jacobs from the 1960s-1970s, preferably one showing her in an urban setting or engaged in community activism.

Image should be:
- Format: JPEG (.jpg)
- Dimensions: Approximately 400x500 pixels (portrait orientation)
- File name: `jane_jacobs.jpg`

Until an image is added, the application will display "Portrait not available" for this character.
